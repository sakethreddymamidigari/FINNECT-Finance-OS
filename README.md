# FINNECT Finance OS 

FINNECT Finance OS is a scalable loan and finance management platform designed for local finance businesses that still rely on paper records. It helps finance owners manage customers, loans, repayments, interest calculations, and transaction history through a secure digital system.

## Features (Planned)
- Finance owner authentication
- Customer management
- Loan management
- Interest calculation
- Payment tracking
- Dashboard and analytics 
- Transaction history
- Role-based access control


## Tech Stack
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT Authentication
